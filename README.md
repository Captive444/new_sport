
# football_analyzer

## Описание

Парсер и анализатор футбольной статистики для анализа команд и игроков.

## Установка


## Использование

- **Сбор данных команд** — позиция в лиге, последние 5 матчей, форма, статистика голов дома/в гостях
- **Сбор данных игроков** — позиция, возраст, рост, полная статистика выступлений
- **Асинхронный парсинг** — Scrapy + Twisted для эффективной обработки
- **Автоматическое логирование ошибок** — повторная обработка упавших команд
- **Структурированное хранение** — данные сохраняются в папки по матчам
